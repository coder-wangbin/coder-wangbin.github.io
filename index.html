🔧 核心工具链

lb CLI + MCP Server

让 AI Agent 能直接操作 K8s 和 GitLab CI 的全套运维工具

  • 11 个 MCP 工具：list_pods / get_logs / diagnose / events / restart / pipeline / job_logs / ingress / tls / status / notify
  • Kuboard SSO 认证（5 步自动完成）+ K8s API 代理
  • Strategic Merge Patch 滚动更新 + 镜像比对跳过
  • 飞书通知：一次 push 一条聚合消息，避免 webhook 重复推送
  • 关联部署（linked_deployments）+ 自动获取 CI tag
GoK8sMCP

dbhub MCP + Git Hook

分支切换自动同步数据库配置，再也不用担心切错库

  • post-checkout hook 检测分支切换 → 读 conf.yml → 转 DSN → 写 dbhub.toml
  • config-watcher 500ms 防抖后热重载连接
  • Go DSN (user:pass@tcp(host:port)/db) → MySQL DSN 自动转换
  • 多项目扩展：<项目名>-current source_id，互不冲突
  • Agent 通过 MCP 始终连接正确数据库
GoMySQLGit

CodeGraph 代码知识图谱

将代码解析成交互式知识图谱，MCP 语义搜索代替 grep

  • 10 个 MCP 工具：context / search / explore / trace / callers / callees / impact / node / files / status
  • 支持 Go / TypeScript / Python / Rust / Java 等 20+ 语言
  • 多分支项目最佳实践：切分支后自动增量同步索引
  • Go 项目特殊配置：框架路由识别、vendor 目录处理
MCPGoTS

  • 踩坑：baseUrl 缺少 /v1、serviceType 错误、模型名错误
